# Guia rápido para o agente

Este diretório contém os arquivos `.md` com as regras que o agente deve seguir ao trabalhar no projeto.

## Arquivos
- `AGENTS.md`: regras globais do repositório
- `backend/AGENTS.md`: regras específicas da API Kotlin/Spring
- `infrastructure/AGENTS.md`: regras específicas de deploy e operação

## Contexto do projeto
- serviço simples
- Kotlin + Spring Boot
- sem banco de dados
- integração direta com API de ERP
- foco em relatórios personalizados
- estrutura com `backend` e `infrastructure`
- deploy inicial no Railway
